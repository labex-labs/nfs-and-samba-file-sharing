Aprenda sobre la compartición de archivos NFS y Samba. Este módulo cubre la Configuración del Servidor NFS, Montaje de Recursos Compartidos NFS, Instalación de Samba, Compartición Pública de Samba, Autenticación de Usuarios de Samba. Dominará estas habilidades esenciales de Linux a través de laboratorios prácticos y desafíos del mundo real.
