Aprenda sobre Compartilhamento de Arquivos NFS e Samba. Este módulo abrange Configuração do Servidor NFS, Montagem de Compartilhamentos NFS, Instalação do Samba, Compartilhamento Público do Samba e Autenticação de Usuário do Samba. Você dominará essas habilidades essenciais do Linux através de laboratórios práticos e desafios do mundo real.
